# Expense Tracker

Track your expenses with categories and charts. Built with React and Firebase.

## Features

- Add and delete expenses
- Categories: Food, Transport, Shopping, Bills, Entertainment, Other
- Filter by category
- Doughnut chart showing spending breakdown
- Real-time sync with Firebase
- Responsive layout

## Tech Stack

- React 17
- Firebase Firestore
- Chart.js
- CSS

## Setup

1. Create a Firebase project and enable Firestore
2. Copy your config to `src/firebase.js`
3. `npm install && npm start`
